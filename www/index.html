Controls:

  • Add node: double-click (hold shape/color keys)
  • Move node: drag
  • Add arrow: shift-drag
  • Delete: select + delete key
  • Node shape: hold 1|3|4|5|6 + double-click
  • Node color: hold Q|W|E|R|T + double-click
  • Node accept state: double-click
  • Subscript: use underscore (S_0)
  • Greek: use backslash (\beta)
  • Pan canvas: middle mouse + drag
